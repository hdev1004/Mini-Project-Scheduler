# Mini Project - Scheduler

### 📕 Schedule Manager

스케쥴 매니저는 일정관리를 편리하게 할 수 있는 웹 사이트 입니다.

Front-End-Framework인 React와 css의 @key_frame 기능을 사용하여 부드러운 효과를 낼 수 있도록 구현했습니다.
